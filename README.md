O que aprendi na aula de hoje, eu descobri que o CSS não é uma linguagem de programação, mas sim uma ferramenta de estilização. Ele funciona selecionando um elemento do HTML e aplicando modificações visuas.A estrutura básica: Aprendi que o código CSS precisa de um seletor (o que eu quero mudar), uma propriedade (o que eu vou alterar, como cor ou tamanho) e um valor (o resultado final, por exemplo, a cor vermelha).
